Design Document: Ranked Guessr Duels

Ranked Guessr duels incorperates your favorite parts of Geoguessr into a fantastical video game environment. Our data uses 5 tables with one JUNCTION table.

ERA

Webpages List

  • connecth.inc.php - Handles player two connecting to a match logic
  • dbh.inc.php - Handles database connection
  • functions.inc.php - Handles functions, primarily relating to verification of login details
  • loginh.inc.php - Handles login logic
  • signuph.inc.php - Handles signup logic
  • login.php - Login Page
  • signup.php - Signup Page
  • accept.php - Handles accepted friend requests
  • reject.php - Handles rejected friend requests
  • admin.php - Handles requests only made by admins (CUD)
  • friend.php - Handles friend sending logic
  • game.php - Main Game, displays connection page, sends match and game data to db
  • home.php - Displays links to main parts of website
  • leaderboard.php - Lists up to top 10 highest ranked in elo players
  • match-end.php - Sends match data after match terminates
  • profile.php - Displays user profiles which contains info on their elo, account-creation date, previous matches, and friends. Defaults to logged in user, with ?id= can refer to other users
  • replay.php - When accompanied with a ?id= refers to a specific replay of a match
  • round-end.php - Sends round data after round terminates
  • signout.php - Destroys session data and logs the user out
  • references.html - Shows references used for project
